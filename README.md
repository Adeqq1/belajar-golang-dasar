# Belajar Golang Dasar

Repositori ini berisi kumpulan kode latihan dan catatan selama proses mempelajari dasar-dasar bahasa pemrograman Go (Golang). Projek ini ditujukan untuk memahami sintaksis, struktur data, dan konsep fundamental dalam ekosistem Go.

## 🚀 Materi yang Dipelajari

Di dalam repositori ini, saya mendokumentasikan beberapa konsep kunci, antara lain:

* **Tipe Data & Variabel:** Penggunaan `string`, `number`, `boolean`, serta deklarasi variabel menggunakan `var` atau `:=`.
* **Struktur Data:** Pemahaman mendalam mengenai **Array**, **Slice**, dan **Map**.
* **Control Flow:** Percabangan (`if`, `switch`) dan perulangan (`for loop`).
* **Function:** Deklarasi fungsi, *multiple return values*, *named return values*, hingga *variadic functions*.
* **Advanced Function:** Penggunaan `closure`, `anonymous function`, dan fungsi sebagai parameter.
* **Defer, Panic, & Recover:** Mekanisme penanganan error dan *cleanup* eksekusi kode.
* **Struct & Interface:** Implementasi pemrograman berbasis objek (OOP) ala Go.
* **Pointer:** Memahami konsep *pass by value* dan *pass by reference*.
